# 🛡️ InsureBot - AI Insurance Assistant - https://insurance-bot-gyvmsu5jerwwuzysr59wa2.streamlit.app/

A conversational AI chatbot for insurance-related queries built with Streamlit and Groq AI.

## Features
- Health, Life, Auto, Home, Travel & Business Insurance Q&A
- File upload support (PDF, DOCX, TXT, CSV)
- Concise & Detailed response modes
- 24/7 deployment via Streamlit Cloud

## Tech Stack
- Python, Streamlit, Groq API (LLaMA 3.3)
